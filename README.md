# Stock-Control

Sistema de gerenciamento de vendas de geladinho, inspirado no negócio da minha família

## Funcionalidades

- Cadastro de sabores com cores personalizadas
- Registro de vendas por sabor
- Reset automático diário por histórico
- Estatísticas de vendas com gastos e lucro

## Tecnologias

- React
- React Router
- React Icons
- CSS puro
- LocalStorage

## Como rodar

```bash
npm install
npm run dev
```
